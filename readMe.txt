#1>. npm commands
  a>. npm
  b>. npm init : create package.json file
  c>. npm install  : to install all modules present in package.json file
  d>. npm install <module name> : install module
  e>. npm install <module name> --save : install module and instructs NPM to include the package inside of the 
                      dependencies section of your package.json automatically, thus saving you an additional step.
  f>. npm install <module name> --save-dev : install as a development dependency
  g>. npm install -g <module name> : install module globally,
